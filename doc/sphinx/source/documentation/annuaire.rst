Créer un annuaire
=================

L'annuaire, à l'instar d'un bottin de type pages d'or, permet de gérer les informations de type coordonnées, adresse, horaire, relatives à une personne ou une organisation.

Les informations créées dans l'annuaire peuvent être affichées au moyen de différents filtres (par thématiques, par type d'activités, par type d'organisations, etc). Les données peuvent être exportées à tout momenet et en fonction des filtres activés. Elles peuvent aussi être consultées sur un plan.

Il est utile de créer une fiche d'annuaire pour toute personne ou organisation que l'on veut renseigner sur le site. Ces données pourront être réutilisées dans le site de différentes manières. Par exemple, on peut afficher les informations encodées dans l'annuaire sur la fiche d'un élu dans la section vie politique.
